# gs-front-orbyn

Repositório de Front-End da Orbyn para a Global Solutions (FIAP).

---

## Sobre

A Orbyn é uma plataforma de análise geoespacial que combina imagens de satélite e inteligência artificial para apoiar o planejamento urbano e rural. O objetivo é dar a governos, produtores rurais e gestores acesso a dados territoriais de forma centralizada e acessível, reduzindo a dependência de processos manuais e fragmentados.

---

## Funcionalidades previstas

- Seleção de regiões no mapa para análise de dados geográficos e ambientais
- Dashboard com indicadores urbanos e agrícolas
- Alertas de risco ambiental
- Recomendações automáticas geradas por IA
- Exportação de relatórios e comparação entre regiões
